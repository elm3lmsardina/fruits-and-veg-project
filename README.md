eshop


A website where admins are able to add, update, and delete items and categories. 
the admins can put every items in a category.
users register and login and access to home and see added all items and see details about them and also can choose which category they want to see items in it.
users can edit their account information.

I created this project for a course.

there is no shopping cart in the website.
